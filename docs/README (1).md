# Documentación del proyecto

Material de referencia para el CRUD de **docentes universitarios** (React + Node.js + MySQL). El código está en las carpetas `client/` y `server/` en la raíz del repositorio.

## Orden sugerido (si no programas o vas paso a paso)

1. **[TUTORIAL_CRUD_DOCENTES.md](TUTORIAL_CRUD_DOCENTES.md)** — Construcción **completa desde cero** (SQL, `server` con npm, React con `create-react-app`, pegar código de cada archivo, arranque). **Un solo archivo para generar la app.** Si ya tienes el repositorio clonado, puedes seguir solo los pasos equivalentes o copiar los archivos del repo.
2. **[TUTORIAL_ARQUITECTURA_CRUD.md](TUTORIAL_ARQUITECTURA_CRUD.md)** — Cómo se conectan las tres capas y qué hace cada archivo principal.
3. **[NARRATIVA_PROBLEMA.md](NARRATIVA_PROBLEMA.md)** — Contexto del caso de la universidad.
4. **[REQUERIMIENTOS_FUNCIONALES_Y_NO_FUNCIONALES.md](REQUERIMIENTOS_FUNCIONALES_Y_NO_FUNCIONALES.md)** — Requisitos que debe cumplir el sistema.
5. **[DISENO.md](DISENO.md)** — Diseño técnico (API, modelo de datos, capas).
